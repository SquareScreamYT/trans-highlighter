# Trans Highlighter

This extension highlights words containing "trans" (not case sensitive) with the trans flag :3

## Display

<img src="colouredtext.gif" alt="Trans-coloured text" height="100px">

## Installation

1. Download the latest VSIX from the release section on GitHub.
2. Go to the extensions tab on the sidebar in VS Code.
3. Click the 3 dots ("Views and More Actions...") at the top right.
4. Click "Install from VSIX".
5. Select the file.
6. Reload the app
7. done :tada:
